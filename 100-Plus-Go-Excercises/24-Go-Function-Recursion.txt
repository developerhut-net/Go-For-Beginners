
24. Go Function Recursion

Goal: Practice recursion (functions calling themselves).

1. Write a recursive function that counts down from 5 to 1.

2. Create a recursive function that calculates the sum of numbers from 1 to n.

3. Write a recursive function that prints "Hello" 3 times.

4. Make a recursive function that finds the factorial of a number.

5. Write a recursive function to print numbers from 1 to 10.
